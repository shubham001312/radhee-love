## 🚀 How to Make it Live (GitHub Pages)

### Step 1 — Upload to GitHub
1. Go to [github.com](https://github.com) → Sign in / Sign up
2. Click **"New repository"** (green button)
3. Name it: `radhee-love`
4. Set it to **Public**
5. Click **"Create repository"**

### Step 2 — Upload the files
1. Click **"uploading an existing file"** link
2. Drag and drop **ALL files** from this folder:
   - `index.html`
   - The entire `images/` folder (all photos)
3. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** tab
2. Scroll down to **"Pages"** (left sidebar)
3. Under **Source** → select **"Deploy from a branch"**
4. Branch: **main** → folder: **/ (root)**
5. Click **Save**
6. Wait ~2 minutes ⏳
7. Your site is live at: `https://YOUR-USERNAME.github.io/radhee-love/`
